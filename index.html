Assignments

Assignment 1 – Basic HTML

For this assignment, I created a website on a classroom-appropriate topic of my choice. The project required me to structure the HTML code correctly, starting with the Doctype declaration and including all necessary elements. I added a main heading (h1) and wrote two detailed paragraphs, each with at least four sentences, to provide content for the page. The website also features two images that are appropriately sized to fit the design. I included a bulleted list with at least three items, a table with two columns and three rows, and two external links to relevant resources. This assignment helped me understand the basics of HTML structure and how to organize content effectively on a webpage.

Assignment 2 – Basic CSS

In this assignment, I got hands-on with CSS to style a simple HTML page. I learned how to change the look and feel of a webpage by adjusting things like colors, fonts, and layout. I practiced using different CSS selectors to target specific elements, and I played around with margins, padding, and borders to better understand how the box model works. This assignment really helped me see how powerful CSS is for making a webpage look polished and visually appealing.

Assignment 3 – Page Layout

In this assignment, the focus was on creating a single webpage that effectively utilizes CSS Flexbox for layout. The webpage was structured using essential HTML5 elements like header, footer, nav, section, aside, and article. A navigation bar was created, and the layout was designed to be responsive, adapting to both large and small screens. Flexbox was used to arrange content into columns, with media queries ensuring a smooth transition to vertical layouts on smaller devices. The assignment also emphasized making the page visually appealing with background colors, padding, and font styles. Images and placeholder text were included to complete the design, following the specified layout diagram.

Assignment 9 – Loops

Assignment 9 demonstrates the use of loops to dynamically generate multiple elements (stars) based on user input. The user enters a number, and the program uses a loop to create that number of stars, placing them randomly on a canvas. If the user enters an invalid number, an error message is displayed using conditional logic.

Assignment 10 – Arrays

In assignment 10,arrays and loops are used to dynamically create a gallery of images. An associative array stores the image paths, titles, and descriptions. A loop iterates over the array, generating the corresponding HTML elements for each image and appending them to the page. When the user clicks on an image, JavaScript accesses the array to retrieve and display the associated title and description above the images.

Assignment 11 – Bird Classes

This assignment focuses on creating an interactive web page using JavaScript, HTML, and CSS. The goal is to display a collection of bird species in a visually engaging layout, with detailed information accessible through a modal dialog. By clicking on each bird, users can view additional details, such as size, lifespan, habitat, and interesting facts
